# SyncShare Browser - Инструкция по сборке

## Быстрый старт

### 1. Установка зависимостей

```bash
cd syncshare-browser
npm install
```

### 2. Запуск в режиме разработки

```bash
npm start
```

### 3. Сборка портативной версии (Windows EXE)

```bash
npm run build:portable
```

Готовый файл появится в папке `dist/` с именем `SyncShare Browser-Portable.exe`.

## Требования для сборки

### Минимальные требования
- Node.js 18 или выше
- npm 9 или выше
- Windows 10/11 (для сборки под Windows)

### Установка Node.js

1. Скачайте установщик с официального сайта: https://nodejs.org/
2. Установите версию LTS (Long Term Support)
3. Проверьте установку:
   ```bash
   node --version
   npm --version
   ```

## Подробная инструкция по сборке

### Шаг 1: Подготовка проекта

Убедитесь, что структура проекта корректна:

```
syncshare-browser/
├── src/
│   └── main.js          # Главный файл приложения
├── extension/           # Папка с расширением SyncShare
│   ├── manifest.json
│   ├── js/
│   ├── css/
│   └── ...
├── package.json
└── README.md
```

### Шаг 2: Установка зависимостей

```bash
npm install
```

Это установит:
- Electron (фреймворк для создания браузера)
- electron-builder (инструмент для сборки в EXE)

### Шаг 3: Настройка (опционально)

#### Изменение стартового URL

Откройте `src/main.js` и измените параметр `homeUrl`:

```javascript
const config = {
  homeUrl: 'https://your-modly-system.local/',  // Ваш URL системы Modly
  // ... другие параметры
};
```

#### Изменение размеров окна

В том же файле можно настроить размеры окна:

```javascript
const config = {
  width: 1400,    // Ширина окна
  height: 900,    // Высота окна
  minWidth: 800,  // Минимальная ширина
  minHeight: 600  // Минимальная высота
};
```

### Шаг 4: Сборка

#### Вариант A: Портативная версия (рекомендуется)

Создает один EXE файл, который можно запускать без установки:

```bash
npm run build:portable
```

**Преимущества:**
- Один файл
- Не требует установки
- Не нужны права администратора
- Можно хранить на флешке

#### Вариант B: Установщик

Создает полноценный установщик:

```bash
npm run build:win
```

**Преимущества:**
- Интеграция в систему
- Ярлыки на рабочем столе
- Автообновления (при настройке)

### Шаг 5: Распространение

После сборки файлы будут в папке `dist/`:

- `SyncShare Browser-Portable.exe` - портативная версия
- `SyncShare Browser Setup X.X.X.exe` - установщик (если собирали)

## Решение проблем

### Ошибка "Cannot find module"

```bash
rm -rf node_modules package-lock.json
npm install
```

### Ошибка при сборке "ENOENT"

Убедитесь, что папка `extension/` существует и содержит расширение:

```bash
ls extension/manifest.json
```

Если файла нет, скопируйте расширение:

```bash
cp -r ../syncshare-extension ./extension
```

### Ошибка "Python not found" (Windows)

Для сборки могут потребоваться инструменты сборки:

```bash
npm install --global windows-build-tools
```

Или установите Visual Studio Build Tools вручную.

### Брандмауэр блокирует соединение

При работе в закрытой сети может потребоваться добавить браузер в исключения брандмауэра.

## Настройка для закрытой сети

### 1. Отключение внешних запросов

Браузер по умолчанию настроен на работу без доступа к интернету.

### 2. Самоподписанные сертификаты

Браузер автоматически принимает самоподписанные сертификаты. Это настроено в `main.js`:

```javascript
acceptSelfSignedCerts: true
```

### 3. Прокси (если требуется)

Для настройки прокси добавьте в `main.js` перед созданием окна:

```javascript
session.defaultSession.setProxy({
  proxyRules: 'proxy.your-network.local:8080',
  proxyBypassRules: '<local>'
});
```

## Обновление расширения

Чтобы обновить встроенное расширение:

1. Скопируйте новую версию расширения в папку `extension/`
2. Пересоберите браузер:
   ```bash
   npm run build:portable
   ```

## Технические детали

### Версии компонентов

- Electron: ^28.0.0
- electron-builder: ^24.9.1
- Manifest расширения: v3

### Архитектура

- **Главный процесс** (`main.js`): Управление окном, загрузка расширения, обработка сертификатов
- **Расширение**: Встраивается через `session.loadExtension()`
- **Рендерер**: Chromium движок от Electron

### Безопасность

- Context Isolation: включен
- Node Integration: отключен
- Sandbox: отключен (требуется для работы расширений)
- Web Security: включен

## Поддержка

При возникновении проблем:
1. Проверьте логи в консоли (DevTools: Ctrl+Shift+I)
2. Убедитесь, что расширение совместимо с Manifest V3
3. Проверьте права доступа к папке проекта
