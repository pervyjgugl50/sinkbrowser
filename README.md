# SyncShare Browser

Защищенный браузер для работы с системой тестирования Modly со встроенным расширением SyncShare.

## Особенности

- **Работа в закрытых сетях**: Поддержка самоподписанных сертификатов и работа без доступа к внешним ресурсам
- **Встроенное расширение**: Расширение SyncShare для сбора статистики и подсказок в системе Modly
- **Не требует аккаунта Google/Yandex**: Работает независимо от браузеров на базе Chromium
- **Портативная версия**: Может быть собран в EXE файл и запускаться без прав администратора
- **Безопасность**: Изолированная среда, контекстная изоляция

## Требования

- Node.js 18+
- npm или yarn

## Установка

```bash
npm install
```

## Запуск в режиме разработки

```bash
npm start
```

## Сборка портативной версии (Windows)

```bash
npm run build:portable
```

После сборки исполняемый файл будет находиться в папке `dist/`.

## Настройка для закрытой сети

### 1. Обновление расширения

Если у вас есть обновленная версия расширения, поместите её в папку `extension/`.

### 2. Настройка URL системы Modly

По умолчанию браузер загружает пустую страницу. Вы можете изменить стартовый URL в файле `src/main.js`:

```javascript
mainWindow.loadURL('https://your-modly-system.local/');
```

### 3. Сертификаты

Для работы с самоподписанными сертификатами в закрытых сетях браузер автоматически принимает их.
При необходимости можно настроить более строгую политику в обработчике `certificate-error`.

## Работа расширения

Расширение SyncShare автоматически активируется на страницах системы Modly:
- `*mod/quiz/attempt.php*` - прохождение тестов
- `*mod/quiz/review.php*` - просмотр результатов
- `*mod/quiz/summary.php*` - сводка по тесту
- `*mod/quiz/view.php*` - страница теста

## Сборка в EXE

Для сборки портативной версии используется electron-builder:

```bash
# Сборка portable версии (один exe файл)
npm run build:portable

# Сборка установщика
npm run build:win
```

Готовые файлы будут в папке `dist/`.
